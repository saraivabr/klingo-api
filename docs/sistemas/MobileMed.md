# MobileMed

Ferramenta que captura radiografias e envia para a nuvem, integrando com o [[Klingo]].

## Funcoes

- Captura de radiografia
- Envio para nuvem
- Integracao com prontuario no [[Klingo]]

## Relacoes

- Pertence a [[Ferramentas Assistenciais]]
- Envia dados para [[Klingo]]
- Redundante com [[LogicMedic]] (mesma funcao, destino diferente)
