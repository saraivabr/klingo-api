# Controle Odonto IRB

Sistema odontologico da [[Ecossistema IRB|IRB]]. Mesmo sistema que o [[Controle Odonto IST]], porem com base de dados separada.

## Funcoes

- Controle de atendimentos odontologicos
- CRM interno
- Agenda odontologica

## Relacoes

- Pertence a [[ERPs]]
- Mesma plataforma que [[Controle Odonto IST]] (bases diferentes)
- Substitui o [[Capim]] (em desuso)
- Redundante com [[Capim]] - ver [[Redundancia de Sistemas]]
