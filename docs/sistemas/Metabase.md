# Metabase

Ferramenta de BI instalada, porem sem dados para alimentar.

## Problema

BI precisa de dados centralizados. Sem [[Sem ERP Unificado|ERP unificado]], nao ha fonte de dados confiavel. Colocar a ferramenta nao soluciona o problema se nao tem processo.

## Relacoes

- Pertence a [[Ferramentas de TI]]
- Depende da centralizacao de dados (PostgreSQL + ElasticSearch)
- Hospedagem futura na [[Hostinger]]
