# Ferramentas de TI

Infraestrutura e comunicacao da [[Ecossistema IRB|IRB]].

## Sistemas

- [[GLPI]] - Abertura de chamados (helpdesk)
- [[Metabase]] - BI instalado mas sem dados para alimentar
- [[Monitoramento]] - Queda de link, servidor, sistemas
- [[Telefonia Fixa]] - Ramal digital SIP (potencial para IA)
- [[Telefonia Movel]]
- [[Meta WhatsApp]] - WhatsApp Business oficial (1 numero)
- [[BR Bots]] - Chatbot atrasado (+1 ano para implementar)
- [[Vsee]] - Telemedicina backup (tecnologia NASA, funciona com 9Kbps)
- [[JotForm]] - Formularios
- [[Hostinger]] - VPS e hosting (hospeda [[Sistema IRB]])

## Observacoes

- [[BR Bots]] sera substituido por IA propria
- [[Telefonia Fixa]] SIP pode ser usada para triagem com IA
- [[Metabase]] precisa de dados centralizados para funcionar
