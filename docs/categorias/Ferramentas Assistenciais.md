# Ferramentas Assistenciais

Ferramentas de apoio ao atendimento medico da [[Ecossistema IRB|IRB]].

## Sistemas

- [[MobileMed]] - Radiografia enviada para nuvem e depois pro [[Klingo]]
- [[LogicMedic]] - Radiografia armazenada localmente (mesma funcao do MobileMed, destino diferente)
- [[Radiologia Dental]] - Equipamentos manuais (pendrive/braçal)
- [[Bionexo]] - Compras via leilao reverso (unico sistema isolado funcional)
- [[Viva Care]] - Controle de ponto dos plantonistas

## Problemas

- [[MobileMed]] e [[LogicMedic]] fazem a mesma coisa (redundancia)
- [[Radiologia Dental]] totalmente manual (pendrive)
- [[Viva Care]] teve fraudes de geolocalizacao no sistema anterior (Pega Plantao)
