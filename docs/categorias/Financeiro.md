# Financeiro

Ferramentas financeiras da [[Ecossistema IRB|IRB]].

## Sistemas

- [[Asaas]] - Modulo de pagamento (futuro: automacao de Pix)
- Excel - Controles manuais de faturamento e conciliacao

## Fluxo

Ver [[Fluxo de Pagamento]] para o fluxo completo.

## Problemas

- [[Processos Manuais]] - Pix confirmado manualmente na recepcao
- Sem chave/input para rastrear pagamento ao paciente/procedimento
- Conciliacao financeira toda em Excel
- Pacientes nao pagam taxa de reserva (alto no-show)
- Se volume aumentar, recepcao nao aguenta validar Pix manualmente
