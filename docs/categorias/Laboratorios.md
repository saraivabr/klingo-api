# Laboratorios

Sistemas de resultados de exames da [[Ecossistema IRB|IRB]]. Cada tipo de exame usa um sistema diferente.

## Sistemas

- [[Hilab]] - Resultados de urina (AMEA)
- [[Worklab]] - Resultados de sangue
- [[EDB]] - Resultados de exames respiratorios

## Fluxo

Todos os resultados precisam ser consolidados manualmente no [[Klingo]]. Se o paciente faz 3 tipos de exame, e necessario entrar em 3 sistemas diferentes.

Ver [[Fluxo de Exames]] para o fluxo completo.

## Problemas

- [[Redundancia de Sistemas]] - 3 sistemas de lab sem integracao
- Cada lab tem seu proprio CRM interno
- Consolidacao manual dos resultados
