# Bionexo

Software de compras com leilao reverso. Unico sistema relativamente isolado e funcional.

## Funcoes

- Requisicao de compras (ex: farmacia com Cintia)
- Leilao reverso entre fornecedores
- CRM embutido

## Relacoes

- Pertence a [[Ferramentas Assistenciais]]
- Usado pelo setor de compras/farmacia
