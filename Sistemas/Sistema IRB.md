# Sistema IRB

Sistema desenvolvido pelo Felipe para substituir/complementar o [[Klingo]]. Em desenvolvimento ha mais de 1 ano.

## Modulos Planejados

### Fase 1 (em andamento)
- Atendimento / Recepcao
- Agendamento
- Painel de controle
- Cadastro de pacientes

### Fase 2
- Estoque
- Financeiro

### Fase 3
- Integracao com SUS
- Laboratorio
- Cirurgico

## Status Atual

- Apenas modulo de atendimento parcialmente funcional
- Cadastro de pacientes com problemas em testes
- Precisa de testes extensivos (tela por tela, perfil por perfil)
- 168 paginas criadas, apenas ~65 funcionais

## Tecnologia

- Hospedado na [[Hostinger]] (VPS)
- Acesso via SSH
- Banco de dados proprio
- Integracao futura com [[Klingo]] via API

## Problemas

- [[Sistema IRB Atrasado]] - +1 ano, 3 modulos contratados, 1 parcial
- Formularios de cadastro muito extensos para recepcao
- Precisa de analista de testes (sugestao de IA para testes automatizados)
- Integracao com [[Klingo]] limitada pela [[API Klingo Limitada]]

## Relacoes

- Pertence a [[ERPs]]
- Hospedado na [[Hostinger]]
- Precisa integrar com [[Klingo]]
- Usado na [[Jornada do Paciente]]
