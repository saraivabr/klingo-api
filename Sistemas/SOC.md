# SOC

Sistema de saude ocupacional. Controla exames admissionais e demissionais.

## Funcoes

- Exames admissionais
- Exames demissionais
- Controle de volume de atendimento para faturamento

## Limitacoes

- Nao tem modulo financeiro
- Vai ate a "beiradinha" do faturamento (volume de atendimentos)
- Precisa de um ERP para completar o processo financeiro

## Relacoes

- Pertence a [[ERPs]]
- Dados fluem para o [[Klingo]]
- Contribui para o [[Fluxo de Pagamento]] (volume → faturamento)
