# ERPs - Sistemas Core

Sistemas principais que sustentam a operacao da [[Ecossistema IRB|IRB]].

> [!warning] Nenhum ERP unificado. Sistemas operam como ilhas isoladas. Ver [[Sem ERP Unificado]].

## Sistemas

- [[Klingo]] - Core medico (prontuario, agendamento, atendimento)
- [[Sistema IRB]] - Em desenvolvimento pelo Felipe (modulos parciais)
- [[Controle Odonto IRB]] - Odontologia base IRB
- [[Controle Odonto IST]] - Odontologia base Dra. Talita (mesmo sistema, base diferente)
- [[Capim]] - Odontologico em desuso (redundante)
- [[SOC]] - Saude ocupacional (admissional/demissional)

## Problemas

- [[Redundancia de Sistemas]] - 3 sistemas odontologicos fazendo a mesma coisa
- [[Sistema IRB Atrasado]] - +1 ano, apenas 1 modulo parcial
- [[API Klingo Limitada]] - Fornecedor nao abre APIs
